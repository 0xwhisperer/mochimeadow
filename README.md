# 🐰 Mochi Meadow

A cozy-but-beatable kawaii 3D adventure in a single HTML file (three.js).

**Play it: https://0xwhisperer.github.io/mochimeadow/**

Hop Mochi the marshmallow bunny across 10 unique sky islands, collect sparkles
before the clock runs out, boop your friends, dodge (or gobble) the grumps,
and face the Grumpapillar in world 10.

- WASD / arrows to move · SPACE to jump · Q/E + scroll for the camera
- 🎮 Full gamepad support (left stick move, A jump, Start pause)
- 📖 In-game illustrated story & guide book
- Saves your run and high score in the browser

Built with three.js — everything (engine, fonts, art) lives in one file.
